## Problem 1:
- T(N) = 2T(N-1) + 1
  - ### Recurrence Relation
    - T(N) = 2[2T(N-2) + 1] + 1
    - T(N) = (2^2) T(N-2)+2+1 = 2^2 [2T(N-3)+1]+2+1
    - T(N) = (2^3) T(N-3)+(2^2) + 2 + 1
    - T(N = (2^k)T(N-k) + (2^(k-1)) + (2^(k-2)) + ...
    - Assume n-k = 0, n = k
    - 2^n + 2^n - 1
    - **O(2^n)**
  - ### Master Method
    - aT(n-b) + f(n)
    - a = 2
    - b = 1
    - f(n) = 1
    - k = 0
    - Since a > 1, T(n) = O(n^0 * 2^n/1) = **O(2^n)**
- T(N) = 3T(N-1) + n
  - ### Recurrence Relation
  - ### Master Method
    - a = 3
    - b = 1
    - f(n) = n
    - k = 1
    - Since a > 1, T(n) = O(n^1 * 3n/3) = O(n^1 * n) = **O(n^2)**
